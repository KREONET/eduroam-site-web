Support

Having trouble with eduroam? We're here to help.

Frequently Asked Questions

How do I set up eduroam for the first time?
Visit the Connect page for instructions on installing the eduroam CAT tool. CAT completes setup automatically in just a few clicks — no manual configuration needed.
I forgot my password.
Please contact the IT helpdesk at {{ site.support.email }} to reset your password.
eduroam was working fine but suddenly gives an authentication error.
If you set up eduroam manually (TOFU), the server certificate may have been renewed. Delete the eduroam Wi-Fi profile from your device and reconnect — this re-trusts the new certificate. Check the server certificate validity period on the Connect page.
I'm connected to eduroam but have no internet.
Try removing the eduroam network from your saved Wi-Fi list and reconnecting. If the issue persists, check that your device is set to obtain an IP address automatically, or contact the helpdesk.
I'm getting a certificate error.
Reinstalling the eduroam CAT profile will automatically renew the certificate. For manual setups, the CA certificate must be installed separately. See the Connect page for details.
How do I connect to eduroam abroad?
If you have the CAT profile installed, your device will connect automatically at any eduroam hotspot worldwide. Check the Coverage page for global availability.
Can I connect multiple devices?
Yes, you can connect multiple devices simultaneously with the same account. Install the CAT profile or complete manual setup on each device.
