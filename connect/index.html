접속 방법

{{ site.t[page.lang].org }} 구성원을 위한 기기별 eduroam 설정 방법을 안내합니다

eduroam CAT (권장)

eduroam CAT(Configuration Assistant Tool)을 사용하면 기기 설정을 자동으로 완료할 수 있습니다. 수동 설정 없이 간편하게 eduroam을 사용하세요.

  1. 1 cat.eduroam.org에 접속하여 {{ site.t[page.lang].org }}를 검색합니다.
  2. 2 운영체제에 맞는 설치 프로그램(프로파일)을 다운로드합니다.
  3. 3 다운로드한 파일을 실행하고 안내에 따라 기관 계정(아이디@{{ site.realm }})과 비밀번호를 입력합니다.
  4. 4 설치 완료 후 Wi-Fi 목록에서 eduroam을 선택하면 자동 접속됩니다.

기관 에듀롬 담당자가 CAT 서비스를 등록한 경우에만 검색 및 이용이 가능합니다.

CAT 도구를 사용하면 인증서 설정이 자동으로 처리됩니다.

  1. 1설정 > Wi-Fi에서 eduroam을 선택합니다.
  2. 2사용자 이름: 아이디@{{ site.realm }}, 비밀번호를 입력합니다.
  3. 3인증서 신뢰 확인 화면에서 신뢰를 탭합니다.

iOS 15 이상에서는 CAT 프로파일 설치를 권장합니다.

  1. 1Wi-Fi 목록에서 eduroam을 선택합니다.
  2. 2EAP 방식: PEAP, Phase 2: MSCHAPV2를 선택합니다.
  3. 3CA 인증서: 시스템 인증서 사용 또는 다운로드한 인증서를 선택합니다.
  4. 4도메인: {{ site.realm }}, ID: 아이디@{{ site.realm }}, 비밀번호를 입력합니다.

Android 11+ 에서는 도메인 입력이 필수입니다.

인증서 정보

CA 인증서를 OS의 키체인에 설치한 경우, 서버 인증서 신뢰 확인 팝업이 뜨지 않으며 인증서 갱신 시에도 별도 조치 없이 자동으로 연결됩니다.
CA 인증서 없이 수동 설정(TOFU)한 경우, 서버 인증서의 유효기간이 끝나면 Wi-Fi 프로파일을 삭제하고 다시 접속해야 합니다.
TOFU(Trust On First Use): 첫 접속 시 사용자가 직접 인증서를 확인하고 신뢰를 부여하는 방식
